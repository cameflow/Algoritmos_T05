Implementación de transformaciones a figuras en QT

Para poder correr el programa se debe de abrir el archivo Tarea05.pro
Esto abrira el proyecto en QT 
Después se utilizara cmd+R o darle en el play para correr el programa

Para utilizar el programa se selecciona primer la figura que se quiere modficar
Después se le debe de dar click en el botón DIBUJAR (Si la figura lo requiere se le debe de indicar el número de lados, el ángulo o el radio dependiendo de el caso para que se dibuje)
Ya que se dibujó la figura se le dará click en las transformaciones y estas se irán relalizando una a una.
Si se quiere hacer una traslación se necesita indicar cuánto se va a trasladar
Si se quiere resetear la figura se le vuelve a dar click en DIBUJAR esto la borrara y si le vovlemos a dar click volverá a aparecer como la primera vez.

El botón de OK o CANCEL cerraran la ventana para que se pueda seleccionar otra figura.
